##The springdoc.api-docs.enabled property determines whether the Springdoc-openapi
## library will expose an endpoint that serves the OpenAPI documentation in JSON format
#springdoc.api-docs.enabled=false

##springdoc.swagger-ui.enabled is a configuration property that controls whether the Swagger UI, which provides an
##interactive API documentation and testing interface, is accessible from your application or not
#springdoc.swagger-ui.enabled=false

###Swagger UI- http://localhost:8080/swagger-ui/index.html#/
###Swagger ui to download the rpi doc:- http://localhost:8080/v3/api-docs.yaml;
